    Download Ollama

    LLM vereist lokale Ollama installatie:

    • Ollama moet draaien op dezelfde machine als de editor (http://localhost:11434).
    • Voor gebruik op afstand (bijv. myeditor.nl), heb je Ollama nodig op de server of een CORS-bypass extensie voor ontwikkeling/testen (niet aanbevolen voor productie).

    Bij online gebruik, start Chrome zonder CORS controle:

    open -na "Google Chrome" --args --disable-web-security --user-data-dir="/tmp/chrome-no-cors"

    Wil je de tool direct proberen zonder installatie? Stuur ons een e-mail via een van onze platformen; we sturen je dan een URL (en inlog) zodat je meteen aan de slag kunt.

    Beschrijf hoe de overgang van scene 1 naar scene 2 verloopt. Dit helpt bij het maken van een vloeiende storyline.

    Voorbeelden van professionele transities:

    • Cut: Directe overgang zonder effect
    • Fade to black: Beeld verdwijnt langzaam naar zwart
    • Cross-fade: Geleidelijke overgang tussen scenes
    • Wipe: Nieuwe scene veegt de oude weg
    • Match cut: Visueel vergelijkbare elementen verbinden scenes
